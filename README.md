# Markdown to PDF
<p>Convert Markdown files to pdf files using JavaScript</p>

### Used libraries <code>marked.min.js</code> and <code>html2pdf.js</code>
<p>The app use <code>marked.min.js</code> library to parse the markdown text and translate it to the preview section and <code>html2pdf.js</code> to generate the PDF.</p>

### How to use it?
#### 1.- Clone the respository
<p>git clone git@github.com:u83mm/markdown_to_pdf_js.git</p>

#### 2.- Launch the app
<ul>
 <li>Double click over <code>index.html</code>.</li>
 <li>Write your markdown code on the left side of the Editor.</li>
 <li>On the right side you can see the preview.</li>
 <li>You can export the preview to a PDF file.</li>
</ul>
